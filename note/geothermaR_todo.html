1 R Rules

1.1 Release name: Main.Sub.Patch for example: 0.0.1

1.2 Function name: package.(noun)(verb)noun

In my opinion,it is important to know the package name when you use R, so let us show it more obviously.

Package Input Via Output Output Functions
geothermaR type verb content type name
ge. sp 2 shp WGS ge.sp2shpWGS
ge. df 2 grid   ge.df2grid
ge. df 2 grid 3D ge.df2grid3D
ge. xy 2 bbox SPDF ge.xy2bboxSPDF
ge.          
ge.
